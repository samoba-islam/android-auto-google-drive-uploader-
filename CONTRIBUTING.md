# Contributing

Thank you for your interest in contributing! This project is licensed under **GPL-3.0**, and all contributions are subject to the same license.

## How to Contribute

1. **Fork** the repository
2. **Create a branch** for your feature or fix:
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. **Make your changes** and commit with clear messages:
   ```bash
   git commit -m "Add: brief description of change"
   ```
4. **Push** to your fork:
   ```bash
   git push origin feature/your-feature-name
   ```
5. **Open a Pull Request** against the `main` branch

## Guidelines

- Follow existing code style and conventions
- Write clear, descriptive commit messages
- Keep PRs focused â€” one feature or fix per PR
- Test your changes on a real device or emulator before submitting

## Reporting Issues

- Use **GitHub Issues** to report bugs or request features
- Include steps to reproduce, expected behavior, and actual behavior
- Attach logs or screenshots if applicable
